# Ilya Sobko Gaming - Организация турниров по Dota 2

![Ilya Sobko Gaming Preview](https://via.placeholder.com/800x400?text=Ilya+Sobko+Gaming+Preview)

## Описание
Ilya Sobko Gaming - это профессиональная организация, специализирующаяся на проведении киберспортивных турниров по Dota 2. Наш сайт предоставляет всю необходимую информацию о предстоящих и текущих соревнованиях, позволяет регистрировать команды и связываться с организаторами.

## Функциональность
- [x] Адаптивный дизайн для всех устройств
- [x] Темная/светлая тема с сохранением в localStorage
- [x] Навигация между страницами
- [x] Фильтрация турниров по статусу (предстоящие, текущие, прошедшие)
- [x] Анимация счетчиков для статистики
- [x] Форма обратной связи с сохранением данных в localStorage
- [x] История посещений страниц
- [x] JS-анимации интерфейса

## Страницы сайта
- `index.html` - Главная страница с анонсами турниров и статистикой
- `about.html` - Страница о компании, ее истории и ценностях
- `tournaments.html` - Полный список турниров с возможностью фильтрации
- `contact.html` - Контактная информация и форма обратной связи
